## 🔨 打包步骤（确保成功）

### 方法一：使用 electron-builder 打包（推荐）

```bash
# 1. 清理旧打包输出
rmdir /s /q release\win-unpacked 2>nul

# 2. 确保依赖已安装
npm install

# 3. 执行打包
npm run dist
```

打包完成后，安装程序在：
- `release/帝意传媒 Setup 1.0.0.exe` ← **这是最终给用户用的安装包**
