## 🔨 打包方法（三选一）

### 方法 A：生成安装程序（推荐，分发给用户用）

**步骤**：
1. 双击 `打包.bat`
2. 等待打包完成（首次 3-5 分钟）
3. 完成后在 `release/` 目录下找到：
   **`帝意传媒 Setup 1.0.0.exe`** ← 这就是安装包！

**验证**：将 `.exe` 文件复制到没有 Node.js 的机器，双击安装，能正常启动说明打包成功。

---

### 方法 B：生成绿色版（ZIP，无需安装）

**步骤**：
1. 双击 `绿色版打包.bat`
2. 等待复制依赖（node_modules 约 200+ 个包，需要几分钟）
3. 完成后在同目录下找到：
   **`帝意传媒-v1.0.0-完整绿色版.zip`** ← 这就是绿色版！

**验证**：将 `.zip` 解压到任意目录，双击 `启动帝意传媒.bat`，能正常启动说明打包成功。

---

### 方法 C：手动打包（最可靠）

**步骤**：
1. 打开 PowerShell（或 CMD）
2. 执行：
```powershell
cd "d:\融媒体发布助手\帝意传媒-mini"
npm install          # 确保依赖已安装
npm run dist         # 运行打包
```
3. 打包完成后在 `release/` 目录下找到 `*.exe` 安装程序。
